# 42sh
This project consists in coding a fully functional shell in C.

By tsisadag, gbourgeo, rfontain, dbaffier

Mandatory part:
- Full edition of commands line
- Redirection and aggregation operators `>` `>>` `<` `<<` `>&` `<&`
- Pipe `|`
- Separators `;`
- Logical operators `&&` `||`
- Built-ins `cd` `echo` `exit` `type`
